How should I cite legume?
-------------------------

If you find Legume to be useful for your research, we would apprecite you citing our paper. For your convenience, you can use the following BibTex entry::

    @article{legume,
      title = {legume},
      author = { ... },
      year = {2020},
      month = feb,
      volume = { ... },
      pages = { ... },
      doi = { ... },
      journal = { ... },
      number = { ... }
    }
